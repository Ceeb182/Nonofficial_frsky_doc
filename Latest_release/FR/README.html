Documents FrSky en Français

Attention :

Ceci n'est pas un site officiel de FrSky.
FrSky n'est pas responsable de l'utilisation des documents présents sur ce site.
FrSky est une marque déposée dont le site officiel est https://www.frsky-rc.com/
FrSky a donné un accord de principe à la mise en place de ce dépôt.
FrSky est informé de toute nouvelle parution de documents sur ce dépôt.
Pour faciliter le contrôle par FrSky :

  • l'arborescence des dossiers est volontairement en Anglais
  • tous les fichiers Readme.md possèdent une section en Anglais
  • toutes les notes de versions comportent une section en Anglais

Définition des versions

  • : traduction la plus fidèle possible du document FrSky
  • : traduction améliorée du document FrSky enrichie d'informations non-officielles
  • : documentation inédite sur un produit ou une procédure

Sommaire

  • Récepteur - D4RII (2-4 voies conventionnelles + RSSI et CPPM en option + 1 Analog + 1 Num / D8 / Portée >1,5km)
  • Récepteur - D8RII Plus (8 voies conventionnelles + 2 Analog + 1 Num / D8 / Portée >1,5km)
  • Récepteur - X8R (8 voies conventionnelles + SPort / D8-D16 / Portée >1,5km / SBUS OUT)
  • Récepteur - X6R (6 voies conventionnelles + SPort + 1 Analog / D8-D16 / Portée >1,5km)
  • Récepteur - X4R X4RSB (3/4 voies conventionnelles + SPort + 1 Analog / D16 / Portée >1,5km / SBUS pour X4RSB / CPPM selon config)
  • Récepteur - S6R S8R (6 ou 8 voies conventionnelles + SPort/ D16 / Portée >1,5km / Gyro+Accel intégré / SBUS+RSSI pour le S8R)
  • Récepteur - G-RX8 (6-8 voies conventionnelles + SPort/ D16 / Portée >1,5km / Fct redondance SBUS IN / SBUS OUT / Altimètre intégré)
  • Récepteur - RX8R (8 voies conventionnelles + SPort / D8-D16 / Portée >1,5km / Fct redondance SBUS IN / SBUS OUT)
  • Récepteur - XSR (Sorties SBUS et CPPM + SPort / D16 / Portée >1,5km)
  • Récepteur - XSR-M (Uniquement SBUS ou CPPM + SPort / D16 / Portée >1,5km)
  • Récepteur - R-XSR (Uniquement SBUS ou CPPM + Sport / D16 / Portée >1,5km / Fct redondance SBUS IN)
  • Récepteur - XM (Uniquement SBUS / D16 / Portée 300m / Pas de télémétrie)
  • Récepteur - XM+ (Uniquement SBUS / D16 / Portée >1,5km / Pas de télémétrie)
  • Récepteur - XMR (6 voies conventionnelles / D16 / Portée 600m / Pas de télémétrie)
  • Récepteur - R9 (8 voies conventionnelles + SPort / R9M / Portée >10km / Fct redondance SBUS IN / SBUS OUT)
  • Récepteur - R9 Slim (6 voies conventionnelles + SPort / R9M / Portée >10km / Fct redondance SBUS IN / SBUS OUT / RSSI / Vcc=3,5~10V)
  • Récepteur - R9 Slim+ (6 voies conventionnelles + SPort / R9M / Portée >10km / Fct redondance SBUS IN / SBUS OUT / RSSI, SPORT inversé, SBUS inversé / Vcc=3,5~12,6V)
  • Récepteur - R9 Mini (4 voies conventionnelles + SPort ou FPort / R9M / Portée >10km / Fct redondance SBUS IN / SBUS OUT)
  • Module d'émission - R9M (Longue portée 868MHz / Format baie JR standard)
  • Module d'émission - R9M Lite (Longue portée 868MHz / Faible latence / Format baie Taranis X-Lite)
  • Capteur - VARI-H (Variomètre précision Haute ou Normale)
  • Capteur - FLVSS (Capteur tension pour batterie LiPo)
  • Accessoire - SCC (Changeur d'Id pour servo SBUS ou capteur SPORT)
  • Accessoire - RB16 (Bus redondant 16 voies pour 2 alimentations et 2 récepteurs + protection contre les surcharges)
